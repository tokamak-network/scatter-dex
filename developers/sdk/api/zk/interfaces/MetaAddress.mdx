# Interface: MetaAddress

A meta-address pair the recipient publishes once.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="spendingkey"></a> `spendingKey` | `string` | Hex secp256k1 private key — keep secret. |
| <a id="viewingkey"></a> `viewingKey` | `string` | Hex secp256k1 private key — keep secret. |
| <a id="metaaddress"></a> `metaAddress` | `string` | Public string. Format: `st:eth:0x<spendingPubCompressed> <viewingPubCompressed>` (66 + 66 hex chars). |
