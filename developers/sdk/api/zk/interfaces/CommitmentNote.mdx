# Interface: CommitmentNote

A commitment note — the secret material backing one escrow entry.

 The full v2 commitment binds the BabyJub signing pubkey
 ([issue #128]):

   commitment = Poseidon(
     TAG_COMMITMENT_V2, ownerSecret, token, amount, salt,
     pubKeyAx, pubKeyAy
   )

 Losing either `ownerSecret` or the EdDSA private key behind
 `pubKeyAx`/`pubKeyAy` makes the funds unspendable. Wallets must
 back both up together.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="ownersecret"></a> `ownerSecret` | `bigint` | - |
| <a id="token"></a> `token` | `bigint` | Token address as uint256. Use `BigInt(addr)` to convert. |
| <a id="amount"></a> `amount` | `bigint` | - |
| <a id="salt"></a> `salt` | `bigint` | - |
| <a id="pubkeyax"></a> `pubKeyAx` | `bigint` | BabyJub signing pubkey x-coordinate. |
| <a id="pubkeyay"></a> `pubKeyAy` | `bigint` | BabyJub signing pubkey y-coordinate. |
