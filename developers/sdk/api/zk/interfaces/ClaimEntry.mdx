# Interface: ClaimEntry

One claim distribution within an authorize proof — "send X of
 the buy token to recipient Y, releasable at time Z, secured by
 this per-claim secret."

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="secret"></a> `secret` | `bigint` | - |
| <a id="recipient"></a> `recipient` | `string` | Ethereum address (0x-prefixed). |
| <a id="token"></a> `token` | `string` | Token address — circuit enforces equality with the order's `buyToken` per used claim. |
| <a id="amount"></a> `amount` | `bigint` | - |
| <a id="releasetime"></a> `releaseTime` | `bigint` | - |
