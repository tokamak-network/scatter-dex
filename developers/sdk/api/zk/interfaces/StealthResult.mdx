# Interface: StealthResult

Output of `generateStealthAddress`.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="stealthaddress"></a> `stealthAddress` | `string` | Checksummed Ethereum address that only the recipient can spend. |
| <a id="ephemeralpubkey"></a> `ephemeralPubKey` | `string` | Compressed ephemeral public key — share with the recipient (typically inside the claim link / receipt) so they can derive the matching private key. |
