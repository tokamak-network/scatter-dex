# Interface: ProveOpts

Optional knobs for a single prove call. Every implementation must
 honor `signal` for cancellation; `onProgress` is best-effort.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="signal"></a> `signal?` | `AbortSignal` | Cancel the proof job. The returned promise rejects with a DOMException("AbortError") when fired. |
| <a id="onprogress"></a> `onProgress?` | (`msg`) => `void` | Called with short status strings ("loading wasm", "running groth16…"). Surfaces are free to ignore. |
