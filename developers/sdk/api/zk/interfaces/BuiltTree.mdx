# Interface: BuiltTree

A built Poseidon Merkle tree. `layers[0]` is the leaf layer
 (padded to `2^depth`); `layers[depth]` is the root layer (one
 element). Suitable input for `getMerkleProof`.

| ------ | ------ |
| <a id="root"></a> `root` | `bigint` |
| <a id="layers"></a> `layers` | `bigint`[][] |
