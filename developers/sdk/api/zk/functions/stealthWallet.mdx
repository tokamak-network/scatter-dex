# Function: stealthWallet()

```ts
function stealthWallet(
   spendingKey, 
   viewingKey, 
   ephemeralPubKeyHex, 
   provider?): Wallet;
```

Convenience: build an `ethers.Wallet` from the derived stealth
 private key. Pass a `provider` if you want to immediately read
 state / send transactions.

## Parameters

| Parameter | Type |
| ------ | ------ |
| `spendingKey` | `string` |
| `viewingKey` | `string` |
| `ephemeralPubKeyHex` | `string` |
| `provider?` | `any` |

## Returns

`Wallet`
