# Function: serializeKeyPair()

```ts
function serializeKeyPair(kp): string;
```

Serialize an EdDSA keypair to a JSON string for storage.
 The private key is hex-encoded; pubkey field elements are
 decimal strings (BigInt-safe).

## Parameters

| Parameter | Type |
| ------ | ------ |
| `kp` | [`EdDSAKeyPair`](../interfaces/EdDSAKeyPair.mdx) |

## Returns

`string`
