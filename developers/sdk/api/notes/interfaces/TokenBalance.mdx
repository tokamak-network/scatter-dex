# Interface: TokenBalance

A token-grouped balance line as the SDK reports it.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="token"></a> `token` | `string` | Token contract address (lowercased). |
| <a id="symbol"></a> `symbol` | `string` | Token symbol used by the stored notes (best-effort). |
| <a id="raw"></a> `raw` | `bigint` | Sum of `note.amount` across unspent notes for this token. |
