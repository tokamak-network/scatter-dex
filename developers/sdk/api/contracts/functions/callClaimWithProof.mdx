# Function: callClaimWithProof()

```ts
function callClaimWithProof(
   signer, 
   settlementAddress, 
   proof, 
inputs): Promise<TransactionResponse>;
```

Send `PrivateSettlement.claimWithProof(...)`. Anyone can submit
 on behalf of the recipient (the address is encoded in the
 public signals), so a relayer can dispatch gaslessly.

## Parameters

| Parameter | Type |
| ------ | ------ |
| `signer` | `Signer` |
| `settlementAddress` | `string` |
| `proof` | [`ClaimProofResult`](../../zk/interfaces/ClaimProofResult.mdx) |
| `inputs` | [`ClaimCallInputs`](../interfaces/ClaimCallInputs.mdx) |

## Returns

`Promise`\<`TransactionResponse`\>
