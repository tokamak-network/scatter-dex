# Function: shortAddr()

```ts
function shortAddr(addr): string;
```

Truncated address helper for display (`0xabcd…1234`). Returns
 empty string for empty input so callers can chain without guards.

## Parameters

| Parameter | Type |
| ------ | ------ |
| `addr` | `string` \| `null` \| `undefined` |

## Returns

`string`
